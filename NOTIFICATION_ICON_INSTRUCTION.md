# Инструкция по созданию иконки уведомлений

Для использования вашей иконки (IMG_1151.PNG) в качестве иконки уведомлений, следуйте этим рекомендациям:

## Требования к иконке уведомлений

1. **Размер**: 24x24 dp (примерно 96x96 px в xxxhdpi)
2. **Формат**: PNG (предпочтительно), не вектор
3. **Фон**: Прозрачный
4. **Цвета**: Лучше использовать однотонную иконку (обычно белую на прозрачном фоне)
5. **Содержание**: Простые геометрические формы, легко узнаваемые в маленьком размере

## Подготовка иконки

1. Откройте IMG_1151.PNG в графическом редакторе
2. Обрежьте изображение до квадрата
3. Упростите изображение до простых форм и линий
4. Убедитесь, что основной элемент иконки помещается в центральный квадрат 12x12 dp (около 48x48 px для xxxhdpi)
5. Удалите фон или сделайте его прозрачным
6. Используйте контрастные цвета для лучшей видимости

## Создание файлов

Создайте файлы иконки в следующих папках:
- drawable-mdpi/ic_notification.png (24x24 px)
- drawable-hdpi/ic_notification.png (36x36 px)
- drawable-xhdpi/ic_notification.png (48x48 px)
- drawable-xxhdpi/ic_notification.png (72x72 px)
- drawable-xxxhdpi/ic_notification.png (96x96 px)

## Использование в коде

Чтобы использовать иконку уведомления в коде, используйте:
```kotlin
val notification = NotificationCompat.Builder(context, CHANNEL_ID)
    .setSmallIcon(R.drawable.ic_notification)  // вместо R.drawable.ic_launcher
    .setContentTitle("Заголовок")
    .setContentText("Текст уведомления")
    .build()
```

## Альтернатива

Если ваше изображение слишком сложное для иконки уведомления, рекомендуется использовать более простую версию или оставить текущую иконку колокольчика (которая уже есть в проекте), но с вашими цветами приложения.

## Обратите внимание

- Иконки уведомлений могут отображаться по-разному в зависимости от версии Android и производителя устройства
- На некоторых устройствах будет отображаться только силуэт иконки
- Для лучшей видимости рекомендуется использовать контрастные цвета и простые формы

## Использование Android Studio

Вы можете снова использовать Android Studio для создания иконки:
1. Щелкните правой кнопкой мыши на папке res
2. Выберите New > Image Asset
3. В разделе "Notification Icons" укажите путь к вашей иконке
4. Следуйте инструкциям мастера
