# Политика конфиденциальности WifiGuard

**Дата последнего обновления:** 12 декабря 2025 г.

## О приложении
WifiGuard — это Android-приложение для анализа безопасности Wi-Fi сетей, разработанное независимым разработчиком Mint1024.

## Основные принципы
- **Полная конфиденциальность**: Все данные хранятся только на вашем устройстве
- **Без передачи данных**: Приложение не отправляет данные на внешние серверы
- **Без аналитики**: Не используются сервисы отслеживания (Google Analytics, Firebase и т.д.)
- **Без рекламы**: Приложение не содержит рекламы

## Сбор данных

### Информация о Wi-Fi сетях
Приложение сканирует и анализирует:
- SSID (название сети)
- BSSID (MAC-адрес точки доступа)
- Уровень сигнала (RSSI)
- Тип шифрования (WEP, WPA, WPA2, WPA3, Open)
- Частота сигнала (2.4 ГГц / 5 ГГц)
- Канал вещания

**Важно:** Эти данные хранятся ТОЛЬКО на вашем устройстве локально.

### Разрешение на местоположение
- **Почему требуется:** Android требует это разрешение для сканирования Wi-Fi сетей (начиная с Android 6.0)
- **Как используется:** Только для получения списка доступных Wi-Fi сетей
- **Что НЕ делается:** Мы НЕ отслеживаем ваше физическое местоположение и НЕ определяем GPS координаты

### История сканирований
- Сохраняется локально на устройстве для отслеживания изменений и обнаружения угроз
- Можно удалить в любой момент в настройках приложения

## Хранение данных
- Все данные хранятся в зашифрованной локальной базе данных (Room Database)
- Используется шифрование AES-256
- Ключи шифрования хранятся в Android KeyStore
- Данные НЕ синхронизируются с облаком
- Данные НЕ передаются третьим лицам

## Использование данных
Собранные данные используются исключительно для:
- Анализа безопасности Wi-Fi сетей
- Обнаружения потенциальных угроз (Evil Twin атаки, небезопасное шифрование)
- Предоставления рекомендаций по улучшению безопасности
- Отображения статистики по сетям
- Фонового мониторинга сетевого окружения (если включено в настройках)

## Передача данных
**Мы НЕ передаем ваши данные:**
- Никакие данные НЕ отправляются на внешние серверы
- Приложение работает полностью офлайн
- Мы НЕ используем аналитические сервисы
- Мы НЕ показываем рекламу
- Мы НЕ передаем данные рекламным сетям

## Ваши права
- **Удалить данные:** Очистить историю сканирований в настройках
- **Отозвать разрешения:** Отключить любые разрешения в настройках Android
- **Удалить приложение:** При удалении все данные автоматически удаляются
- **Отключить фоновый мониторинг:** Можно отключить в настройках приложения
