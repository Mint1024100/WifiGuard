## Матрица тестирования совместимости

### Android (AOSP)
| Версия | API | Критичные зоны проверки |
|---|---:|---|
| 8.0 / 8.1 | 26–27 | старт, разрешения, уведомления (каналы), WorkManager |
| 9 | 28 | ресиверы/сканирование, миграции Room |
| 10 | 29 | ограничения Wi‑Fi сканирования, кэш, стабильность UI |
| 11 | 30 | фоновые ограничения, уведомления |
| 12 | 31 | PendingIntent, фоновые ограничения |
| 13 | 33 | `POST_NOTIFICATIONS`, `NEARBY_WIFI_DEVICES`, доступ к SSID/BSSID |
| 14 | 34 | запрет старта FGS из фона, `FOREGROUND_SERVICE_LOCATION` (манифест), сценарии запуска |
| 15 | 35 | регрессия относительно 14 |
| 16 | 36 | при наличии SDK/targetSdk — регрессия и новые ограничения |

### OEM / ROM
| Прошивка | Фокус |
|---|---|
| HyperOS (Xiaomi) | battery optimization, автозапуск, уведомления |
| One UI (Samsung) | спящий режим, фоновые ограничения |
| ColorOS/RealmeUI/OxygenOS | агрессивный kill фоновой работы |
| HarmonyOS 1.0–4.2 | фоновые политики, уведомления |
| LineageOS | близко к AOSP, проверить ресиверы/сканирование |
| GrapheneOS/CalyxOS//e/OS | приватность/ограничения доступа к Wi‑Fi данным |

### Минимальный smoke-набор
- Первый запуск → выдача разрешений → сканирование → показ результатов.
- Уведомление об угрозе (вкл/выкл звук/вибрация).
- Перезапуск устройства → проверка планирования фоновых задач (если прошивка позволяет).
