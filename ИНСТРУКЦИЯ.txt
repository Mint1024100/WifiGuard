ВАЖНО! Удалите старое приложение перед установкой нового!

═══════════════════════════════════════════════════════════════

Вариант 1 - Через командную строку:
───────────────────────────────────

1. Откройте терминал в папке проекта

2. Выполните команды:

   adb uninstall com.wifiguard
   ./gradlew clean assembleDebug
   ./gradlew installDebug

3. Запустите приложение на устройстве


Вариант 2 - Через Android Studio:
──────────────────────────────────

1. Удалите приложение WifiGuard с устройства вручную

2. В Android Studio:
   - Build → Clean Project
   - Build → Rebuild Project
   
3. Нажмите Run (зелёная кнопка)


Вариант 3 - Вручную:
─────────────────────

1. Удалите приложение с телефона

2. Соберите APK:
   ./gradlew clean assembleDebug

3. APK будет в:
   app/build/outputs/apk/debug/app-debug.apk

4. Установите APK на устройство


═══════════════════════════════════════════════════════════════

✅ ПРОВЕРКА УСПЕХА:
