## OEM / Custom ROM: известные особенности и рекомендации

### Xiaomi / Redmi / POCO (MIUI / HyperOS)
- **Проблема**: агрессивная оптимизация батареи, ограничения автозапуска, “заморозка” фоновой работы и уведомлений.
- **Рекомендации**:
  - Настройки → Приложения → WifiGuard → Батарея → **Без ограничений**
  - Настройки → Автозапуск → включить WifiGuard
  - Настройки → Уведомления → WifiGuard → включить все пункты

### Samsung (One UI)
- **Проблема**: “спящий режим” приложений, фоновые ограничения, частичная деградация уведомлений.
- **Рекомендации**:
  - Настройки → Приложения → WifiGuard → Батарея → **Не оптимизировать**
  - Добавить в “Неотслеживаемые приложения” (если доступно)

### OPPO / Realme / OnePlus (ColorOS / RealmeUI / OxygenOS)
- **Проблема**: очень агрессивная оптимизация батареи; возможен срыв периодических задач.
- **Рекомендации**:
  - Отключить оптимизацию батареи для WifiGuard
  - Разрешить автозапуск/работу в фоне

### Huawei / Honor (HarmonyOS 1.0–4.2 на базе Android)
- **Проблема**: дополнительные системные политики фоновой работы; иногда нестабильные уведомления/ресиверы.
- **Рекомендации**:
  - Настройки → Приложения → WifiGuard → Батарея → Запуск приложения → отключить автоуправление → включить всё

### vivo / iQOO
- **Проблема**: “фоновые ограничения” и iManager/оптимизация.
- **Рекомендации**:
  - Настройки → Батарея → разрешить фоновую активность
  - iManager → автозапуск: разрешить WifiGuard

### LineageOS / PixelExperience
- Обычно близко к AOSP.
- Риск: у части сборок могут быть изменены политики геолокации/сканирования.

### GrapheneOS / CalyxOS / /e/OS
- **Риск**: усиленные политики приватности могут ограничивать доступ к Wi‑Fi данным/идентификаторам.
- Рекомендация: проверять работу при включённых ограничениях приватности.

### HarmonyOS NEXT (5.0)
- **Не поддерживает APK** — приложение не может быть установлено.
