# Инструкция по замене иконки приложения

Для полной замены иконки приложения на вашу иконку (C:\Users\asus\Downloads\Telegram Desktop\IMG_1151.PNG), выполните следующие шаги:

## 1. Подготовка изображений

Вам нужно создать несколько версий иконки в разных разрешениях:

1. Используйте графический редактор или онлайн-сервис (например, Android Asset Studio) для создания иконок.

2. Подготовьте следующие файлы на основе вашей IMG_1151.PNG:

### Для традиционных иконок (до Android 7.1):
- mipmap-mdpi/ic_launcher.png (48x48 px)
- mipmap-hdpi/ic_launcher.png (72x72 px)
- mipmap-xhdpi/ic_launcher.png (96x96 px)
- mipmap-xxhdpi/ic_launcher.png (144x144 px)
- mipmap-xxxhdpi/ic_launcher.png (192x192 px)

### Для адаптивных иконок (Android 8.0+):
Создайте отдельные слои:

#### Фоновый слой (background):
- drawable/ic_launcher_background.xml (уже создан)
- Или подготовьте изображение: mipmap/ic_launcher_background.png

#### Передний слой (foreground):
- mipmap/ic_launcher_foreground.png (основное изображение для адаптивной иконки)

## 2. Как подготовить изображения:

1. Откройте IMG_1151.PNG в графическом редакторе
2. Обрежьте до квадрата (рекомендуется 1080x1080 пикселей)
3. Для адаптивной иконки поместите изображение в центр области 660x660 пикселей (это 66dp из 108dp)
4. Создайте версии для разных плотностей, используя масштабирование:
   - mdpi: 48px (1080 * 48 / 108)
   - hdpi: 72px (1080 * 72 / 108)
   - xhdpi: 96px (1080 * 96 / 108)
   - xxhdpi: 144px (1080 * 144 / 108)
   - xxxhdpi: 192px (1080 * 192 / 108)

## 3. Копирование файлов:

Скопируйте подготовленные файлы в следующие папки проекта:

```
app/src/main/res/
├── drawable/
│   ├── ic_launcher_background.xml (уже создан)
│   └── ic_launcher_foreground.xml (уже создан)
├── mipmap-anydpi-v26/
│   ├── ic_launcher.xml (уже создан)
│   └── ic_launcher_round.xml (уже создан)
├── mipmap-mdpi/
│   └── ic_launcher.png (ваша иконка)
├── mipmap-hdpi/
│   └── ic_launcher.png (ваша иконка)
├── mipmap-xhdpi/
│   └── ic_launcher.png (ваша иконка)
├── mipmap-xxhdpi/
│   └── ic_launcher.png (ваша иконка)
├── mipmap-xxxhdpi/
│   └── ic_launcher.png (ваша иконка)
└── mipmap/
    └── ic_launcher_foreground.png (ваша иконка)
```

## 4. Проверка:

После добавления изображений соберите проект и проверьте, как выглядит новая иконка.

## Важные замечания:

- Убедитесь, что изображения имеют квадратную форму
- Для адаптивных иконок (Android 8.0+) изображение должно быть разделено на два слоя: фон и передний план
- Используйте непрозрачные изображения для традиционных иконок, или учтите, что Android будет использовать белый фон для адаптивных иконок

## Альтернативный способ:

Вы также можете использовать Android Studio:
1. Щелкните правой кнопкой мыши на папке res
2. Выберите New > Image Asset
3. Выберите "Launcher Icons (Adaptive and Legacy)"
4. В поле "Path" укажите путь к вашей иконке (C:\Users\asus\Downloads\Telegram Desktop\IMG_1151.PNG)
5. Настройте параметры и нажмите "Next"
