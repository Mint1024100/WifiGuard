# Документация WifiGuard

Добро пожаловать в документацию по WifiGuard — приложению для анализа безопасности Wi-Fi сетей.

## Обзор

WifiGuard — это Android-приложение, которое анализирует доступные Wi-Fi сети на предмет потенциальных угроз безопасности, помогая пользователям избежать подключения к небезопасным сетям.

## Технологии

- **Язык**: Kotlin
- **Архитектура**: Clean Architecture + MVVM
- **UI**: Jetpack Compose
- **DI**: Hilt
- **БД**: Room
- **Фоновые задачи**: WorkManager
- **Сборка**: Gradle с Kotlin DSL
