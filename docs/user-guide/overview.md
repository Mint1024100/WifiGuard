# Руководство пользователя WifiGuard

## Обзор функций

WifiGuard — это приложение для анализа безопасности Wi-Fi сетей, которое помогает пользователям избежать подключения к небезопасным сетям и получать уведомления о потенциальных угрозах.

## Основные сценарии использования

### 1. Проверка безопасности новой сети

**Сценарий**: Пользователь находится в общественном месте и хочет подключиться к Wi-Fi

**Шаги**:
1. Откройте приложение и перейдите на экран "Сканер"
2. Нажмите "Сканировать" для поиска доступных сетей
3. Просмотрите список сетей с указанием уровня угрозы
4. Выберите сеть с безопасным уровнем (зеленый индикатор)
5. Избегайте сетей с высоким уровнем угрозы (красный индикатор)

### 2. Проверка текущей сети

**Сценарий**: Пользователь хочет проверить безопасность текущей сети

**Шаги**:
1. Откройте экран "Сканер"
2. Нажмите "Сканировать"
3. Найдите текущую сеть в списке
4. Оцените уровень угрозы и тип шифрования
5. Ознакомьтесь с рекомендациями по безопасности

### 3. Мониторинг безопасности (в фоне)

**Сценарий**: Пользователь хочет получать уведомления при подключении к небезопасным сетям

**Шаги**:
1. Перейдите в "Настройки"
2. Включите "Автоматическое сканирование"
3. Настройте интервал сканирования
4. Включите "Уведомления об угрозах"
5. При подключении к небезопасной сети придет уведомление

## Права доступа и их использование

Приложение запрашивает следующие разрешения:

### Обязательные разрешения
- **Местоположение (ACCESS_FINE_LOCATION)**: Для сканирования Wi-Fi сетей (требование Android 6+)
- **Wi-Fi статус (ACCESS_WIFI_STATE)**: Для получения информации о Wi-Fi сетях
- **Изменение Wi-Fi (CHANGE_WIFI_STATE)**: Для управления подключениями (в будущем)

### Дополнительные разрешения
- **Уведомления (POST_NOTIFICATIONS)**: Для оповещения об угрозах (Android 13+)
- **Соседние Wi-Fi устройства (NEARBY_WIFI_DEVICES)**: Для сканирования без привязки к местоположению (Android 13+)
- **WAKE_LOCK**: Для фонового мониторинга
- **ПЕРЕЗАГРУЗКА**: Для возобновления мониторинга после перезагрузки

## Рекомендации по использованию

### Безопасное поведение
1. **Не подключайтесь к открытым сетям** без необходимости
2. **Предпочитайте сети с WPA2/WPA3 шифрованием**
3. **Избегайте сетей с WEP или слабым WPA шифрованием**
4. **Будьте осторожны с сетями с подозрительными именами**
5. **Используйте VPN** при подключении к общественным сетям

### Настройка приложения
1. Включите автоматическое сканирование для постоянной защиты
2. Настройте уведомления для получения своевременных предупреждений
3. Регулярно проверяйте историю сканирований
4. Используйте темную тему при необходимости

### Интерпретация результатов

#### Уровни угрозы
- **Безопасно (зеленый)**: Сеть защищена современным шифрованием
- **Низкий риск (голубой)**: Сеть с устаревшим, но работоспособным шифрованием
- **Средний риск (желтый)**: Сеть с потенциальными уязвимостями
- **Высокий риск (оранжевый)**: Сеть с известными уязвимостями
- **Критический риск (красный)**: Открытая сеть или с серьезными уязвимостями

#### Типы безопасности
- **WPA3**: Наиболее безопасный тип
- **WPA2/WPA3**: Совместимый режим, безопасный
- **WPA2**: Безопасный, но устаревший
- **WPA**: Устаревший, потенциально уязвимый
- **WEP**: Небезопасный, устаревший
- **Открытая**: Нет шифрования
