# Быстрый старт с WifiGuard

## Установка и настройка

### Необходимые компоненты

- Android Studio Hedgehog (2023.1.1) или новее
- JDK 17 или выше
- Устройство с поддержкой Wi-Fi или эмулятор с поддержкой Wi-Fi

### Сборка проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Mint1024100/WifiGuard.git
   cd WifiGuard
   ```

2. Убедитесь, что все зависимости установлены:
   ```bash
   ./gradlew check
   ```

3. Соберите проект:
   ```bash
   # Для Debug сборки
   ./gradlew assembleDebug
   
   # Для Release сборки
   ./gradlew assembleRelease
   ```

4. Установите приложение:
   ```bash
   ./gradlew installDebug
   ```

## Запуск в Android Studio

1. Откройте проект в Android Studio
2. Выберите `app` в качестве модуля для запуска
3. Выберите устройство (реальное или эмулятор)
4. Нажмите кнопку запуска (зеленый треугольник)

## Первоначальная настройка

При первом запуске приложения:

1. **Разрешения**: Приложение запросит необходимые разрешения:
   - Местоположение (для сканирования Wi-Fi)
   - Уведомления (для оповещения об угрозах)

2. **Настройка автоматического сканирования**:
   - Перейдите в "Настройки"
   - Включите "Автоматическое сканирование" если необходимо
   - Настройте интервал сканирования (5 мин, 15 мин, 30 мин, 1 час)

3. **Настройка уведомлений**:
   - Включите "Уведомления об угрозах"
   - Настройте звук и вибрацию по необходимости

## Основные функции

### Сканирование Wi-Fi

1. Откройте экран "Сканер"
2. Нажмите кнопку "Сканировать"
3. Приложение покажет список доступных Wi-Fi сетей
4. Каждая сеть будет проанализирована на безопасность

### Анализ безопасности

1. На экране "Анализ" отображается:
   - Общий уровень риска
   - Количество проанализированных сетей
   - Количество безопасных сетей
   - Найденные угрозы
   - Рекомендации по безопасности

### Мониторинг в фоне

Приложение автоматически:
- Мониторит подключения к Wi-Fi сетям
- Отправляет уведомления при подключении к небезопасным сетям
- Выполняет периодическое сканирование (если включено)

## Проверка работоспособности

После установки убедитесь, что:

1. Приложение успешно запускается
2. Отображаются доступные Wi-Fi сети
3. При подключении к сети показывается информация о ней
4. Функция анализа безопасности работает корректно
5. Уведомления приходят (если включены)

## Устранение типичных проблем

### Приложение не запускается

- Проверьте, что используется JDK 17 или выше
- Убедитесь, что Android Studio имеет актуальные SDK
- Попробуйте выполнить `./gradlew clean` и повторную сборку

### Не видит Wi-Fi сети

- Проверьте, что разрешения местоположения предоставлены
- Убедитесь, что Wi-Fi включен на устройстве
- Попробуйте перезапустить приложение

### Не получает уведомления

- Проверьте настройки уведомлений приложения в системе
- Убедитесь, что разрешение на уведомления предоставлено
- Проверьте настройки в "Настройки" → "Уведомления"
